# Chapitre 0

## Installation de `create-next-app`

Afin de créer votre application React, je vous recommande de passer par [Next.js](https://nextjs.org) qui offre des fonctionnalités en plus que l'on détaillera dans un prochain chapitre.

Pour installer `create-next-app`, vous devez l'installer avec `npm`, `yarn`, `pnpm` ou `bun`. Dans ce cours, nous utiliserons `yarn` mais libre à vous d'utiliser le gestionnaire de paquets qui vous convient le mieux.

Pour installer `create-next-app` avec `yarn`, voici la commande à exécuter :

```bash
yarn global add create-next-app
```

## Création d'une application React avec `create-next-app`

Afin de créer une application React avec `create-next-app`, vous devrez exécuter la commande suivante :

```bash
yarn create next-app --use-yarn < nom du projet >
```

**_Vous devez remplacer_** `< nom du projet >` **_par le nom de votre projet_**
