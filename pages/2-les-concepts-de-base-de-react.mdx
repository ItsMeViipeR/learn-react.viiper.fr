# Chapitre 2

Dans ce chapitre, nous allons voir les différents concepts de base de React qui seront détaillé dans les chapitres suivants

## Les composants

Dans le développement d'une application React, nous utilisons des `composants`. Un composant est un éléments que l'on affiche à l'utilisateur mais qui n'est pas dans le fichier de notre page.

## Les états (states)

React utilise des states que l'on peut récupérer (`get`) et modifier (`set`). Ces states stockent des données de tous types.

## Les propriétés (props)

Pour bien développer une application React, il faut maitriser les `props`. Ces `props` permettent de transmettre des données d'un composant à un autre.
