# Chapitre 1

## Le JSX, c'est quoi ?

Le JSX est le type de fichiers utilisé par React. Le JSX est une extension de syntaxe utilisée principalement avec React pour décrire la structure des interfaces utilisateur.
Il permet d'écrire du code HTML de manière déclarative directement dans des fichiers JavaScript, facilitant ainsi la création et la manipulation d'éléments d'interface de manière efficace et intuitive.

## Pourquoi le JSX ?

React utilise le JSX car, comme dit précédement, le JSX permet d'écrire nativement du HTML dans le code JavaScript, pour une plus grande facilité de développement.
