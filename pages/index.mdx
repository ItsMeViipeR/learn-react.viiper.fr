# Learn React

## Introduction

Bienvenue dans ce cours dédié à React. Dans ce cours complet sur React, nous verrons comment débuter avec React jusqu'au déploiment de votre première application React hébergée sur [Vercel](https://vercel.com).

## React, c'est quoi ?

React est une librarie JavaScript permettant de créer des sites web complets. Développé et maintenu par Facebook, React doit sa popularité à son approche innovante du développement web.

## Pourquoi apprendre React ?

Que vous soyez un développeur web débutant cherchant à créer un site web ou un développeur web chevronné, React est fait pour vous. Grâce à son approche _component-first_, c'est à dire que tous les élements sont des composants,
le développement d'une application avec React est simple et intuitive si l'on a les bonnes pratiques. Bonnes pratiques que l'on verra dans ce cours.

## Composition de Learn React

Ce cours sera divisé en différents chapitres, chacun sur une notion précise. Les chapitres peuvent donc êtres très long à lire pour des notions complexes.

Au fil des chapitres, nous explorerons les principes fondamentaux de React tels que les composants, le JSX, le rendu conditionnel, la gestion des états, et bien plus encore.

Nous aborderons également des sujets avancés tels que les hooks, le routage, et la gestion des effets, vous préparant ainsi à construire des applications web robustes et performantes.
